# Saint Thomas College Website

Este repositorio contiene el c\u00f3digo de un sitio web est\u00e1tico para el Colegio Saint Thomas Valpara\u00edso.

## Descripci\u00f3n
El proyecto consiste en una \u00fanica p\u00e1gina HTML (`index.html`) acompa\u00f1ada de una carpeta `img` con las im\u00e1genes utilizadas en el dise\u00f1o. Todo el c\u00f3digo est\u00e1 escrito en HTML, CSS y JavaScript sin dependencias externas.

## Dependencias
No se requieren dependencias adicionales; basta con un navegador web moderno para visualizar el sitio.

## Despliegue
1. Clona este repositorio:
   ```bash
   git clone <url-del-repositorio>
   ```
2. Entra al directorio del proyecto:
   ```bash
   cd web
   ```
3. Inicia un servidor web est\u00e1tico. Por ejemplo, usando Python:
   ```bash
   python3 -m http.server
   ```
   Esto servir\u00e1 el sitio en `http://localhost:8000`.

Tambien puedes publicar los archivos en cualquier servicio que permita hospedar p\u00e1ginas est\u00e1ticas, como GitHub Pages o un servidor web tradicional.
